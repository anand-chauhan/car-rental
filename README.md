🚖 Taxi Booking & Car Rental Website

A modern and responsive Taxi Booking & Car Rental Web Application that allows users to explore available cars, view pricing, and book rides easily. This project focuses on providing a smooth and user-friendly experience for customers looking to rent vehicles.

🔗 Live Demo: [View Website](https://anand-chauhan.github.io/car-rental/index.html)

🌟 Features

🚗 Browse multiple cars (SUVs, Sedans, Hatchbacks)

📅 Easy booking process (date & location selection)

💰 View pricing for different vehicles

🔍 Clean and user-friendly UI

📱 Fully responsive design

⚡ Fast and lightweight performance

The platform offers a wide range of vehicles and flexible booking options for different travel needs.

🛠️ Tech Stack

HTML – Structure

CSS – Styling & layout

JavaScript – Functionality & interactions
